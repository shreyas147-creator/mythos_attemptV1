## 🎯 Success Metrics

### **You're on track if:**

✅ Training loss decreasing smoothly  
✅ Throughput >400 tokens/sec/GPU  
✅ Expert utilization >80%  
✅ Value network MSE <0.05  
✅ No expert collapse  
✅ GPU memory usage stable  
✅ No frequent OOM errors  
✅ Checkpoints saving successfully  
✅ Benchmarks improving each eval  

### **Red flags:**

❌ Loss not decreasing after 10K steps  
❌ Throughput <300 tokens/sec/GPU  
❌ Expert utilization <60%  
❌ Frequent OOM errors  
❌ NaN/Inf in gradients  
❌ Expert collapse (some experts never used)  
❌ Value network MSE >0.1  
❌ Communication timeouts
